This project presents a comprehensive end-to-end image classification pipeline on the CIFAR-10 dataset. The study compares two distinct modeling approaches:

Baseline Model: A Convolutional Neural Network (CNN) trained from scratch
Transfer Learning Model: MobileNetV2 pretrained on ImageNet, adapted for CIFAR-10 classification
The objective is to evaluate which methodology provides superior generalization on unseen data and to quantify the performance difference using standard evaluation metrics.# DeepLearningCNN
